# WIP-DoctrineIndex

This folder holds the working plan for turning `docs/doctrine.md` into a stable
project foundation.

The goal is to keep the doctrine stack easy to navigate:

- `docs/doctrine.md` as the front door
- `docs/wu-wei-method.md` as the flow doctrine
- `docs/cloudflare-quota.md` as the quota and polling guardrail
- `docs/project-memory.md` as the durable context ledger
- `docs/architecture.md` and `docs/state-machine.md` as system truth

The detailed milestone plan lives in `plan.md`.
